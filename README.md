# Basic Concepts of Python for Beginners.

# Variable:- In Python, a variable is a container for storing a value.
In Python, there are four different types of variables: int, float, str, and bool.

# Data Types:- Python has several built-in data types, including strings, integers, and floating-point numbers. 
Python has several built-in data types, including numeric types (int, float, complex), string (str), boolean (bool), and collection types (list, tuple, dict, set).
